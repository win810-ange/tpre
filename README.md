>> l'application ecolomboko est une plateforme de sensibilisation contre la destruction de l'environnemen

>>> fonctionnalité 
>> user  : dans l'application user on peut creer des utilisateur et connecter les utilisateurs afin qu'il soit capable de faire de signalement, participer au forum de discussion autour de l'environnement et voir les statistique

>> signalement : dans l'application signalement on peut signaler les differents destruction de l'environnement et faire les statistique en terme de type [ "pollution", "erosion", "inondation"] et les commune de la ville de kinshasa

>> forum ; dans les forums on a des sujets et des reponses que chaque utilisateur peut enregistrer

>> on peut acceder aux modifications de suppression ou modification des sujets, signalement et user en se connectant en tant que administrateur nom-de-domaine.com/admin et en passant le formulaire
